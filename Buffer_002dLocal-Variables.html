3 Buffer-Local Variables

All buffer-local variables in this package are provided so that the expansion of certain forms can be configured file-locally. This chapter specifies them.

Variable: for-binder ¶

This buffer-local variable has a symbol as its value that is used as the head of certain let-like forms, which defaults to pcase-let. The function cell (see (elisp)Function Cells) of the symbol stores either a macro or a special form compatible with the following grammar: 

(for-binder ([(form expression)…])
  body…)

Above, for-binder is the symbol, the form of form is specified by the macro or the special form, an expression is an expression, and bodys are the body. As an example, a semantics similar to in-ref clauses of cl-loop forms can be achieved by setting for-binder to cl-symbol-macrolet: 

for-binder
     ⇒ cl-symbol-macrolet



(let ((list (list 1 2 3)))
  (for-do ((reference (for-in-list list))
           (:do (setf reference nil))))
  list)
     ⇒ (nil nil nil)
